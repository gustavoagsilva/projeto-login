# 🔐 Projeto Login

## 📌 Descrição do Projeto

O **Projeto Login** é uma aplicação web simples que apresenta uma interface de autenticação de usuário. O sistema permite que o usuário insira suas credenciais (como e-mail ou usuário e senha) em um formulário de login.

O objetivo principal do projeto é demonstrar a construção de uma **tela de login moderna e funcional**, utilizando tecnologias básicas do desenvolvimento web.

A interface foi desenvolvida com foco em organização visual e usabilidade, simulando um sistema de autenticação comum em aplicações web.

---

## 🎯 Objetivo do Projeto

O objetivo deste projeto é praticar conceitos fundamentais do **desenvolvimento front-end**, especialmente a criação de formulários e interfaces de autenticação.

Durante o desenvolvimento foram trabalhados conceitos como:

- Criação de formulários de login
- Estruturação semântica com HTML
- Estilização de interface com CSS
- Organização de layout e elementos visuais
- Estruturação de projetos web

Este tipo de interface é amplamente utilizado em sistemas reais para **controle de acesso de usuários**.

---

## 🚀 Funcionalidades

O projeto inclui as seguintes funcionalidades:

- Interface de **login de usuário**
- Campos para inserção de **usuário ou e-mail**
- Campo para **senha**
- Botão para envio do formulário
- Layout organizado e intuitivo
- Estrutura preparada para integração com sistemas de autenticação

---

## 🛠 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5** → Estrutura da página e formulário de login
- **CSS3** → Estilização da interface
- **JavaScript** → Manipulação de eventos e interações do formulário
- **Git e GitHub** → Controle de versão e armazenamento do projeto
